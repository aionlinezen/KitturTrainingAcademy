# SSC Funnel Backend

Render-hosted backend powering multi-tenant SSC funnels:
- Razorpay payments
- Resend emails
- Cloudinary uploads
- JWT admin login
- In-memory stores (swap to DB later)

## Dev
- `npm run dev` (nodemon)
- Set env in Render dashboard

## Deploy
- Render Web Service with env vars (see .env.example)
